Clustering data using Principal Component Analysis and K-Means

This panel shows how multidimensional data can be reduced to 2, 3 readable dimensions. This technique can be useful in reducing the number of variables present in the dataset. We aim to choose the fewest components from the dataset, while preserving as much of the original information as possible. K-means is an algorithm used to establish new, redefined, clustered based on the distance of the points in the new 2D dataset. This is useful for better defining the outliers and how different the data points lie in the set.
